\section{Cara Menghubungkan Python Dengan PostgreSQL}

\subsection{Definisi PostgreSQL}
\cite{momjian2001postgresql}PostgreSQL adalah database server dengan basis open source yang paling maju. PostgreSQL sendiri sudah dipakai oleh banyak kalangan di luar negeri karena faktor keamanan yang cukup terjamin. Seperti database server lainnya, PostgreSQL memakai bahasa pada umumnya yaitu standard SQL tetapi beberapa syntax diubah untuk kepentingan keamanan.Dalam PostgreSQL dapat menyimpan titik geografis dari aplikasi Sistem Informasi Geografis yang membuat bahasa ini popular dalam kalangan programmer Sistem Informasi Geografis. Selain titik geografis, PostgreSQL dapat menyimpan objek geometric diantaranya titik, garis, dan area. Selain itu, PostgreSQL dapat beroperasi seperti layaknya bahasa lainnya seperti manipulasi data atau membuat table.
